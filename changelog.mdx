<Update label="2024-10-15" description="v1.2.0" tags={["feature", "improvement"]}>

## New Features

- Added real-time collaboration for team editing sessions.
- Introduced version history with diff viewer.

## Improvements

- Optimized page rendering speed by `>30%`.
- Enhanced MDX component support for custom icons.

## Bug Fixes

- Fixed parsing errors with nested `{code blocks}`.
- Resolved mobile navigation glitches.

```javascript
// Example: New collaboration API
import { collaborate } from '@shihabsalah/docs-sdk';

const session = await collaborate({ docId: 'your-doc-id', users: ['user1', 'user2'] });
```

</Update>
